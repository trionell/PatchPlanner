Round 3 changes:
  • Sources are one node with a compact row per source (mirrors how Channels/Mixer already scale) — solves the height problem at 32 sources.
  • Cable paths are now computed from each port dot's real rendered position (same technique the actual canvas uses via getBoundingClientRect), so they always terminate exactly on the handles, including after rows are added/removed.
  • Color has one source of truth — the Channel. A Source (and any Processing/Stagebox port between it and its channel) simply inherits: one channel's color if everything downstream agrees, grey the moment two reachable channels disagree. No color field anywhere else.

Everything from rounds 1–2 (Channels/Stagebox-Multi sections, splitter billing, direct-to-channel patching, DI as a plain Processing device) still stands. Say the word and I'll turn this into the spec.
